Big Horn Sheep

Toronto instrumental trio Big Horn Sheep blend elements of funk, jazz, electronica, and rock into an innovative, all-original repertoire.

Approach

A website for Toronto instrumental trio Big Horn Sheep, this project required bold imagery and graphic design, unique typography, the presentation of band audio tracks and images, as well as performance dates and contact information.

  •  Down

Solution

The design solution consisted of a Single Page Application (SPA) approach using AngularJS and Bootstrap 4. It features animated page entrances/exits, embedded soundcloud audio players, a pinterest style image gallery, Bootstrap modal popups for show dates with band posters (also of mason media design), with unique banner images and branding. The site was also an exercise in Atomic CSS, with an effort to limit styles to native Bootstrap 4 classes.

  •  Down

Tech Specs

The site is built with HTML5, CSS3, Bootstrap 4, and AngularJS. It also required UX/UI design, copywriting, graphic design, color palette development, image/logo creation, and a Facebook page design.
